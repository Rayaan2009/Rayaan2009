## About

I'm a **Year 12 student at Doha Academy** and an **AI + Software Engineering student building real-world systems that combine software, hardware, and intelligent technologies.**

I like problems that sit where code meets the physical world, such as voice-controlled smart homes, sensor-driven security systems, and small tools that make everyday work faster. My focus is on clean engineering, understanding how things work from first principles, and shipping projects that actually run.

- Targeting a **Computer Science / Artificial Intelligence** degree
- Studying towards **A Levels** in Maths, Physics, Chemistry, and Computer Science
- Interested in **AI/ML, IoT & robotics, embedded systems, and cybersecurity**
